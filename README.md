Цілі проєкту

1. Аналіз онлайн-кампаній (Facebook & Google Ads)

1.1 Отримати середнє, максимум і мінімум щоденних витрат окремо для Google та Facebook

1.2 Визначити 5 днів із найбільшим загальним ROMI (Google + Facebook)

1.3 Знайти кампанію з найвищим тижневим value

1.4 Визначити кампанію з найбільшим приростом охоплення (reach) місяць-до-місяця

1.5 Знайти назву та тривалість найдовшого безперервного (щоденного) показу adset_name (об’єднані дані Google + Facebook)

⸻

2. Аналіз поведінки користувачів та сесій у GA4

2.1 Підготувати дані про події користувачів та сесій у GA4

2.2 Розрахувати конверсії у розрізі дат і каналів трафіку

2.3 Порівняти конверсії між різними посадковими сторінками (landing pages)

2.4 Перевірити кореляції між залученістю користувачів та покупками
